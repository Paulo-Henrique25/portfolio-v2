# Portfólio v2 - Paulo Henrique

Portfólio pessoal desenvolvido com React e Vite para apresentar minha trajetória, habilidades, projetos e formas de contato como desenvolvedor Front-End.

## Sobre o projeto

Este projeto foi criado para substituir uma versão anterior mais simples do meu portfólio, trazendo uma interface mais moderna, responsiva e profissional.

O objetivo do portfólio é apresentar minhas habilidades, projetos desenvolvidos, tecnologias que utilizo e canais de contato para oportunidades como Desenvolvedor Front-End Júnior, Assistente de Desenvolvimento ou projetos freelancer.

## Funcionalidades

- Layout moderno e responsivo
- Header fixo com navegação por seções
- Seção inicial de apresentação
- Seção sobre minha trajetória
- Lista de tecnologias e habilidades
- Seção de serviços
- Cards de projetos com imagem, tecnologias, link do projeto e link do código
- Seção de contato com e-mail, LinkedIn, GitHub e WhatsApp
- Footer dinâmico com ano atual

## Tecnologias utilizadas

- React
- Vite
- JavaScript
- CSS

- GitHub

## Projetos apresentados

### Union Polls

Aplicação de enquetes desenvolvida com foco em interface responsiva, organização visual e experiência do usuário.

### Dublador de Idiomas

Projeto de estudo desenvolvido durante treinamento, com foco em JavaScript, consumo de API de tradução, reconhecimento de voz e síntese de fala.

### Conversor de Moedas

Projeto desenvolvido para praticar lógica com JavaScript, manipulação de valores e construção de interface simples e funcional.

### Jokenpô

Jogo de pedra, papel e tesoura desenvolvido para praticar lógica de programação, eventos no JavaScript e manipulação do DOM.

## Como executar o projeto

Clone este repositório:

```bash
git clone https://github.com/paulo-henrique25/portfolio-v2.git
```

Acesse a pasta do projeto:

```bash
cd portfolio-v2
```

Instale as dependências:

```bash
npm install
```

Rode o projeto localmente:

```bash
npm run dev
```

Gere a versão de produção:

```bash
npm run build
```
